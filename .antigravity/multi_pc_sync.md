# Multi-PC Sync Instructions for Antigravity

This directory contains the context files for features implemented or planned in this project. Use the prompt below to initialize a new session when switching between computers.

---

## 📋 Session Initialization Prompt

Copy and paste this prompt when starting a new chat session with Antigravity on a different machine:

```text
I am pair programming with you across two PCs. I've pulled the latest changes from Git. Please read the files inside the `.antigravity/` folder to understand the current features implemented on this device (`star_feature.md` and `uptrend_pullback_feature.md`), as well as our future ideas (`feature_ideas.md`). Going forward, when we implement new features that generate markdown plans or walkthroughs, save those files in `.antigravity/` and add them to the list of changes to be committed so that both machines stay in sync.
```
